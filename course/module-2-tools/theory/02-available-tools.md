## 2. Бесплатные и дешевые варианты

На данный момент (начало 2026 г.) лучший опыт в agentic development дают дорогие Claude модели. Выгоднее использовать Claude по подписке (`$2` / `$100` / `$200`), чем по API. На тарифах за `$100` и `$200` лимиты достаточно большие. Claude Code очень удобный CLI, через который можно запускать параллельно множество агентов.   

Однако, если вы не готовы платить $100/мес, есть альтернативы (хотя пользовательский опыт будет хуже).

### 2.1. OpenCode — бесплатный CLI-агент

**Что это:** Open-source CLI-агент, альтернатива Claude Code. 103K+ stars на GitHub, активное развитие.

**Плюсы:**
- **Бесплатный** — платите только за API провайдера (Claude, GPT, Gemini)
- **75+ провайдеров** — не привязаны к одной компании, можете переключаться
- **Ollama поддержка** — запускайте модели локально совсем бесплатно
- **Privacy** — не хранит ваши данные, BYOK
- **Гибкость** — Terminal UI, Desktop приложение, IDE extension

**Что можно:**
- Те же задачи, что и Claude Code (автономная разработка, multi-file changes, коммиты)
- Использовать дешевые модели (DeepSeek R1 за $0.70/1M input токенов вместо Claude за $3)
- Бесплатные модели от OpenCode (MiniMax) — полностью бесплатно, но будете упираться в лимиты
- Локальные модели через Ollama — полностью бесплатно, но требуется хорошее железо для хостинга модели

---

### 2.2. Claude Code Guest Pass

Попросите Guest Pass у коллег (с подпиской Max)

**Guest Pass:**
- **7 дней Claude Pro** — получаете от пользователя с Max подпиской ($200/мес)
- **Unlimited Sonnet 4.5** — нет лимитов на сообщения
- **Opus 4.6 доступ** — самая умная модель для сложных задач

---

### 2.3. Ollama — локальные модели

**Что это:** Инструмент для запуска open-source моделей локально на вашем компьютере. Данные не уходят в интернет, все бесплатно.

**Поддерживаемые модели:**
- **Qwen Coder** (7B-480B) — coding и multimodal от Alibaba
- **Phi-4** (14B) — state-of-the-art для малых моделей от Microsoft
- **Mistral** (7B-123B MoE) — open-source GPT альтернатива

**Системные требования:**
- **GPU:** Nvidia/AMD с 8+ GB VRAM (для 8B моделей), 48+ GB для 70B
- **RAM:** 16+ GB
- **Диск:** 10-100 GB в зависимости от модели
- **Альтернатива:** CPU-only режим (медленнее в 10-20 раз)

**Что можно:**
- **Privacy-критичные задачи** — коммерческий код, NDA проекты
- **Unlimited использование** — платите только за электричество
- **Эксперименты** — пробуйте разные модели бесплатно
- **Обучение** — разбирайтесь как работают модели

**Чего нельзя:**
- **Качество как у Opus/GPT-5** — локальные модели хуже на сложных задачах
- **Большие контексты** — ограничены GPU памятью (обычно 8K-32K токенов)
- **Мгновенные ответы** — генерация медленнее облачных API (зависит от железа)

**Ограничения:**
- Нужно техническое понимание (установка драйверов, настройка)
- Потребление энергии (GPU под нагрузкой = 200-350W)
- Качество ниже frontier моделей на 20-40% (зависит от задачи)

---

### 2.4. Kilo Code — open-source VS Code расширение

**Что это:** open-source расширение для VS Code

**Что можно:**
- Те же задачи что и Cursor, но платите напрямую провайдеру
- Комбинировать модели (планирование на Opus, реализация на DeepSeek)
- Использовать Ollama для бесплатной работы

**Ограничения:**
- Нужна настройка (выбор провайдера, API ключи)
- Меньше интеграций чем у коммерческих продуктов

---

### 2.5. Google Antigravity — бесплатная multi-agent IDE

**Что это:** Экспериментальная IDE от Google с multi-agent orchestration.

**Плюсы:**
- **Бесплатно** — generous rate limits на Gemini 3
- **Multi-agent** — несколько агентов работают параллельно (editor, terminal, browser)
- **Async tasks** — агент работает в фоне, вы делаете другое
- **Claude и GPT-OSS** — не только Gemini

**Что можно:**
- Экспериментировать с multi-agent подходом
- Параллельная разработка (один агент пишет код, другой тесты)
- Интеграция с Google сервисами

**Ограничения:**
- Много багов

---

### 2.6. Codex CLI — официальный CLI от OpenAI

**Что это:** CLI-агент от OpenAI, входит в ChatGPT подписки ($8+ в месяц).

**Особенности:**
- **Multimodal** — работает со screenshots, GIF, WebP
- **MCP support** — Model Context Protocol для интеграций
- **Web search** — агент ищет информацию в интернете
- **GPT-5.3-Codex** — 25% быстрее стандартного GPT-5.2

**Тарифы:**
- **Go** ($8/мес) — базовый доступ
- **Plus** ($20/мес) — стандартный
- **Pro** ($200/мес) — unlimited, GPT-5 Pro, приоритет

**Когда использовать:** Если уже платите за ChatGPT, получаете CLI бесплатно. Enterprise-фокус (MCP, безопасность).

---

### 2.7. DeepSeek — дешевая альтернатива для exploratory задач

**Что можно:**
- **Exploratory coding** — пробуете разные подходы, не боясь потратить бюджет
- **Черновая работа** — генерация тестов, документации, рефакторинг
- **Массовая обработка** — 1000 файлов на $5 вместо $150
- **Self-hosting** — скачать модель и запустить локально (бесплатно), найти провайдера с бесплатным доступом

---

### 2.8. Kimi CLI — бесплатный агент от Moonshot AI

**Что это:** Бесплатный CLI-агент от Moonshot AI. Поддерживает несколько моделей, включая Kimi K2.

**Плюсы:**
- **Бесплатно** — щедрые лимиты без подписки
- **Мультимодельность** — поддержка нескольких моделей
- **CLI** — работает в терминале, как Claude Code

**Когда использовать:** Как бесплатную альтернативу для exploratory задач и прототипирования.

---

### 2.9. Комбинирование инструментов

> [!TIP]
> Никто не использует один инструмент для всего. Эффективная стратегия — комбинировать под задачи.

**Типичные комбинации:**

**$0-20/мес:**
```
Чат:      DeepSeek (бесплатно)
IDE:      Kilo Code + VS code
CLI:      OpenCode + DeepSeek API ($5-10/мес) или Kimi CLI (бесплатно)
Локально: Ollama + Qwen Coder (бесплатно)
```

**$40-60/мес:**
```
Чат:      ChatGPT Plus ($20) для обучения
IDE:      Cursor Pro ($20)
CLI:      Claude Code Pro ($20)
Локально: Ollama для экспериментов
```

**Профессионал ($100-200/мес):**
```
Чат:      Claude / ChatGPT
IDE:      Cursor Pro ($20)
CLI:      Claude Code Max 5x ($100/мес) или Max 20x ($200/мес)
Локально: Ollama для privacy-critical
```
