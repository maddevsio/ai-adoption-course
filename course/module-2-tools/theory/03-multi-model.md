## 3. Модели и мульти-модельная стратегия

### 3.1. Главный принцип: модель под роль, а не "лучшая модель на все"

Искать "лучшую модель" и использовать её везде — это Ferrari для поездки в магазин.

**Правильный подход:** разбить задачу на части, назначить каждой подходящую модель, сделать ревью умной моделью.

### 3.2. Три категории моделей

> [!NOTE] Цены на модели обновляются часто. Актуальные цены: [anthropic.com/pricing](https://anthropic.com/pricing), [openai.com/pricing](https://openai.com/pricing). Данные ниже актуальны на февраль 2026.

#### 🧠 Умные модели — Планирование и архитектура

**Модели:**
- Claude Opus 4.6 — $5 / $25 за 1M токенов
- GPT-5.2 Pro — $21 / $168 за 1M токенов
- Gemini 3 Pro — $2 / $12 за 1M токенов (до 200K контекст)

**Когда использовать:** Планирование, глубокое размышление
- Проектирование архитектуры системы
- Разбиение крупной задачи на подзадачи (декомпозиция)
- Код-ревью критичного кода (security, performance)
- Исследование незнакомого codebase
- Написание спецификаций
- Планирование рефакторинга большой кодовой базы

> [!NOTE]
> Дорогие модели не дают пропорционально лучший результат на простых задачах. Если задача — переименовать переменную, Opus и Haiku дадут одинаковый результат, но Opus будет медленнее и дороже.

---

#### ⚡ Средние модели — Планирование в цикле, реализация, ревью

**Модели:**
- Claude Sonnet 4.5 — $3 / $15 за 1M токенов
- GPT-5.2 — $1.75 / $14 за 1M токенов
- Gemini 3 Pro (стандартный контекст) — $2 / $12 за 1M токенов

**Когда использовать:** Реализация фич по готовой спецификации
- Планирование внутри агентского цикла (Ralph Loop)
- Код-ревью обычного кода
- Рефакторинг модулей
- Написание тестов
- Debugging сложных багов
- Итеративная разработка (agent в Claude Code/Cursor)

---

#### 💨 Дешевые/быстрые модели — Массовая реализация

**Модели:**
- Claude Haiku 4.5 — $1 / $5 за 1M токенов
- GPT-5.2 Instant — включено в $1.75 / $14
- Gemini 3 Flash — ~$0.50 / $3 за 1M токенов
- DeepSeek V3 — $0.30 / $1.20 за 1M токенов

**Когда использовать:** Работа по минимальным изолированным задачам

- Автогенерация тестов по шаблону
- Batch-обработка (сотни файлов)

## 4. Как выбирать: задача → инструмент

### 4.1. Блок-схема выбора

```
Ваша задача?
│
├─ Простой вопрос / обучение / объяснение
│  └─ LLM-чат (ChatGPT, Claude.ai, DeepSeek)
│
├─ Автодополнение во время набора кода
│  └─ IDE-копилот (GitHub Copilot)
│
├─ Написать функцию/модуль (до 500 строк)
│  ├─ Простая задача → Чат или Копилот
│  └─ Сложная задача → AI-IDE (Cursor, Kilo Code) или CLI-агент
│
├─ Реализовать фичу (несколько файлов, тесты, интеграция)
│  └─ CLI-агент (Claude Code, OpenCode)
│
├─ Рефакторинг большой кодовой базы
│  └─ CLI-агент с умной моделью (Opus/GPT-5 Pro)
│
├─ Массовая генерация (тесты, документация, миграция)
│  └─ CLI-агент с дешевой моделью (DeepSeek, Haiku) + Batch API
│
└─ Privacy-критичная задача / работа офлайн
   └─ Ollama (локальные модели)
```

### 4.2. Примеры: Задача → Инструмент → Модель

#### Простые задачи

- **"Как работает JWT?"**
  **Инструмент:** ChatGPT / Claude.ai | **Модель:** любая (Sonnet, GPT-5.2) | **Стоимость:** $0 | **Время:** 1 мин

- **Автодополнение при написании функции**
  **Инструмент:** GitHub Copilot | **Модель:** GPT-5.x | **Стоимость:** $10/мес (unlimited) | **Время:** мгновенно

#### Средние задачи

- **Написать API endpoint для регистрации**
  **Инструмент:** Cursor Agent Mode | **Модель:** Sonnet 4.5 | **Стоимость:** ~$0.50 | **Время:** 5 мин

- **Добавить JWT аутентификацию в проект**
  **Инструмент:** Claude Code CLI | **Модель:** Sonnet 4.5 | **Стоимость:** ~$1.50 | **Время:** 10 мин

- **Спроектировать микросервисную архитектуру**
  **Инструмент:** Claude.ai или Claude Code | **Модель:** Opus 4.6 | **Стоимость:** ~$2 | **Время:** 15 мин

#### Масштабные задачи

- **Сгенерировать тесты для 50 файлов**
  **Инструмент:** Aider CLI + скрипт | **Модель:** DeepSeek R1 | **Стоимость:** ~$0.30 | **Время:** 20 мин

- **Рефакторинг legacy системы (20K LOC)**
  **Инструмент:** Claude Code CLI | **Модель:** Opus 4.6 (план) + Sonnet 4.5 (код) | **Стоимость:** $8-12 | **Время:** 1-2 часа

- **Миграция Python 2 → 3 (100 файлов)**
  **Инструмент:** Aider CLI Batch | **Модель:** DeepSeek R1 Batch | **Стоимость:** $2-5 | **Время:** 4-8 часов

#### Приватность

- **Анализ конфиденциального кода**
  **Инструмент:** Ollama | **Модель:** DeepSeek R1 локально | **Стоимость:** $0 | **Время:** зависит от GPU

## Выводы

1. **Комбинирование > одиночный инструмент.** Чат для обучения, копилот для автодополнения, CLI-агент для автономных задач.

2. **Мульти-модельная стратегия обязательна.** Opus для планирования, Sonnet для реализации, DeepSeek для массовых задач. Экономия 50-80%.

3. **Бесплатные инструменты полноценны.** OpenCode + DeepSeek API + Ollama покрывают 90% задач за $5-10/мес.

4. **Задача определяет инструмент.** Простой вопрос → чат. Автодополнение → копилот. Фича → CLI-агент.
