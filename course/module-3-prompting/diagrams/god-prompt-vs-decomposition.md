# Антипаттерн: God-Prompt vs Декомпозиция

God-Prompt — попытка описать всё в одном промпте.

```mermaid
flowchart LR
    GP["12 требований<br>в одном промпте"] --> R1["Поверхностно<br>пропуски, несогласованность"]

    style R1 fill:#d33,stroke:#333,color:#fff
```

Декомпозиция — каждый шаг решает одну задачу на нужной глубине.

```mermaid
flowchart LR
    D1["Архитектура"] --> D2["Модели"] --> D3["Интеграции"] --> D4["API"] --> D5["Тесты"] --> D6["Метрики"]
    D6 --> R2["Глубокая реализация"]

    style R2 fill:#2d6,stroke:#333,color:#fff
```

**Правило:** если промпт длиннее 10-15 строк — скорее всего, это God-Prompt. Разбейте на шаги.
