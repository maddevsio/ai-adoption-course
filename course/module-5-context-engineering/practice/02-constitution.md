# Шаг 1: Конституция проекта (10 мин)

### Цель

Создать конституцию проекта — "память" агента, которая сохраняется между сессиями.

### Шаблон

Используйте файл [templates/constitution-template.md](../templates/constitution-template.md) как основу.

### Задание

1. Откройте шаблон конституции
2. Выберите свой проект (или создайте учебный)
3. Заполните все секции с помощью агента:
   - **Project Overview:** название, назначение, статус
   - **Tech Stack:** используемые технологии с версиями
   - **Architecture Decisions:** 1-2 ключевых решения (например, "почему выбрали PostgreSQL", "почему монолит, а не микросервисы")
   - **Patterns:** 2-3 обязательных паттерна (например, "все handlers возвращают Result<T, Error>")
   - **Antipatterns:** 2-3 запрета (например, "не использовать var, только let/const")
   - **Conventions:** правила именования, структура файлов
4. Сохраните конституцию в отдельный файл, например `docs/constitution.md`

### Как сделать конституцию обязательной для агентов

Конституция — это **не** AGENTS.md. AGENTS.md — краткая справка (tech stack, команды запуска, базовые правила). Конституция — полный документ с ADR, паттернами, антипаттернами (~400-500 строк), который слишком велик для AGENTS.md.

Чтобы агент гарантированно прочитал конституцию, добавьте в AGENTS.md инструкцию:

```markdown
## Перед началом работы

**ОБЯЗАТЕЛЬНО** прочитай docs/constitution.md перед любой задачей.
```

Другие способы:
- **[GitHub Spec Kit](https://github.com/github/spec-kit)** хранит конституцию в `.speckit/constitution.md` и автоматически подключает её через skills
- **Claude Code hooks** — можно настроить pre-tool hook, который напоминает агенту прочитать конституцию
- **Skills** — встроить чтение конституции в первый шаг skill (как в [cc-sdd](https://github.com/gotalab/cc-sdd))

### Проверка

Попросите агента прочитать конституцию и объяснить ключевые правила:

```
Прочитай конституцию. Какие паттерны обязательны для этого проекта?
Какие antipatterns запрещены? Кратко перечисли основные правила.
```

Если агент правильно пересказал ключевые моменты — конституция работает.
