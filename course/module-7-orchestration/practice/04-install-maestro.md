# Установите Maestro

Maestro — десктопное приложение для оркестрации AI-агентов. Поддерживает Claude Code, Codex, OpenCode.

### Краткий обзор UI

Maestro предоставляет графический интерфейс для управления агентами:
- **Sessions** — список активных и завершённых сессий с агентами
- **Agent Panel** — окно взаимодействия с агентом (аналог терминала, но с визуальным интерфейсом)
- **File Explorer** — просмотр файлов проекта и изменений, внесённых агентом

### Поддержка платформ

- **macOS:** полная поддержка (основная платформа)
- **Linux:** поддерживается, возможны ограничения в UI
- **Windows:** через WSL2

---

### Упражнение: установка и настройка (5 мин)

**1. Установите Maestro:**

Инструкция для вашей ОС: https://docs.runmaestro.ai/getting-started

**2. Подключите агента:**

Убедитесь, что хотя бы один агент (Claude Code, Codex или OpenCode) установлен и работает в терминале. Maestro использует их как провайдеров — своих моделей у него нет.

**3. Откройте проект:**

Создайте новую сессию в Maestro, указав директорию вашего проекта.

### Проверка

- [ ] Maestro установлен и запускается
- [ ] Агент-провайдер подключён, сессия открыта
- [ ] Можете отправить сообщение агенту через Maestro
