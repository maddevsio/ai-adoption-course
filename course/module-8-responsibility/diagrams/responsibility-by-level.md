# Ответственность по уровням

Чем больше автономности у агента — тем выше ваша ответственность за настройку и контроль. На каждом уровне добавляется новая зона ответственности.

```mermaid
flowchart LR
    subgraph L3["Level 3\nОдин агент"]
        R3["Ваша ответственность:\n- Ревью кода\n- Запуск тестов\n- Проверка diff\n- Co-authorship"]
    end

    subgraph L4["Level 4\nАгент + MCP"]
        R4["Добавляется:\n- Аудит MCP-конфигов\n- Ограничение доступов\n- Логирование действий\n- Credentials isolation"]
    end

    subgraph L5["Level 5\nОркестрация"]
        R5["Добавляется:\n- Координация агентов\n- Разрешение конфликтов\n- Мониторинг системы\n- Граница: стратегия\nза человеком"]
    end

    L3 --> L4 --> L5
```

**Ключевой принцип:** код от ИИ = ваш код. На всех уровнях финальная ответственность за результат — на человеке. Агент — это инструмент, не субъект ответственности.
