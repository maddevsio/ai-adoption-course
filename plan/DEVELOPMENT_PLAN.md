# План разработки курса

## Фазы

### Фаза 1: Подготовка (задачи 01-03)

Цель: собрать фактуру, утвердить инструменты и примеры.

- Собрать актуальные данные по инструментам (цены, фичи, статус)
- Подготовить кейсы с метриками для мотивационного модуля
- Определить конкретные версии инструментов, MCP-серверов, моделей

Результат: фактура для всех модулей собрана, решения зафиксированы.

### Фаза 2: Теоретические материалы (задачи 04-11)

Цель: написать текст теоретической части каждого модуля.

8 задач — по одной на модуль. Большинство задач независимы, можно выполнять параллельно.

Результат: 8 MD-файлов с теорией, готовых к ревью.

### Фаза 3: Практические материалы (задачи 12-19)

Цель: создать инструкции, упражнения, тестовые задания, тестовый проект.

8 задач. Зависят от Фазы 1 (версии) и частично от Фазы 2 (содержание теории).

Результат: пошаговые инструкции, упражнения, шаблоны, тест, тестовый проект.

### Фаза 4: Видео-демо (задача 21)

Цель: подготовить сценарии для скринкастов. Запись — ручная работа.

Результат: сценарии для 4-5 видео-демо.

### Фаза 5: Ревью и сборка (задачи 22-23)

Цель: проверить, собрать, протестировать.

Результат: готовый курс.

## Зависимости

```
Фаза 1 (подготовка)
  ├── Фаза 2 (теория) — частично параллельно
  │     Задачи 06, 08, 11 — можно сразу, без зависимостей
  │     Задачи 04, 05, 07, 09, 10 — после Фазы 1
  ├── Фаза 3 (практика) — после Фаз 1 и 2
  └── Фаза 4 (видео) — после Фаз 2+3
Фаза 5 (ревью) — после всех
```
