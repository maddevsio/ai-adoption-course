# Задача 21: Сценарии видео-демо

## Цель

Подготовить сценарии для 4-5 скринкастов, дополняющих текстовые материалы.

## Что сделать

### Видео 1: Установка и первый запуск (5-7 мин)

Модуль 2. Установка Claude Code CLI, настройка API-ключа, первый запрос, архитектурная схема проекта.

### Видео 2: Работа с агентом — полный цикл (5-7 мин)

Модуль 4. Задача → промпт → агент работает → ревью → корректировка.

### Видео 3: SDD в действии (5-7 мин)

Модуль 5. Спецификация → агент реализует → сверка с criteria.

### Видео 4: Настройка MCP (5-7 мин)

Модуль 6. Установка серверов → конфигурация → задача с MCP.

### Видео 5: Параллельные агенты (5-7 мин)

Модуль 7. Git worktree → 2 агента → слияние.

### Для каждого видео подготовить

- Тайминги (что в какую секунду)
- Что на экране
- Ключевые фразы для озвучки
- Используемые инструменты и примеры из практических заданий

## Формат результата

MD-файл `course/video-scripts.md`

## Критерии приёмки

- 4-5 сценариев
- Каждый 5-7 мин
- Используют те же инструменты и примеры, что в практике
- Достаточно деталей для записи

## Зависимости

- Задачи 14, 16, 17, 18, 19 (практические материалы)
