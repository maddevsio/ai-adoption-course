# Задача 22: Ревью и согласованность

## Цель

Проверить все материалы на согласованность, полноту и качество.

## Что проверить

1. **Согласованность**: термины, уровни зрелости, названия инструментов — одинаковы везде
2. **Полнота**: каждый теоретический блок → практика; нет "висящих" тем
3. **Прогрессия**: модуль N строится на модуле N-1, нет скачков
4. **Качество текста**: конкретика, без AI-slop, единый тон
5. **Актуальность**: цены, версии, ссылки
6. **Техническая корректность**: код работает, команды корректны, шаблоны рабочие
7. **Хронометраж**: каждый модуль укладывается в заявленное время

## Формат результата

MD-файл `course/review-report.md`

## Зависимости

Все задачи 04-21.
