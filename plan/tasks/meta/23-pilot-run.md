# Задача 23: Пилотный прогон

## Цель

Протестировать курс на 2-3 добровольцах. Собрать обратную связь, исправить проблемы.

## Что сделать

1. Выбрать 2-3 добровольца (уровень 1-2)
2. Каждый проходит курс самостоятельно, фиксирует:
   - Где застрял
   - Реальное время на каждый модуль
   - Что лишнее / что не хватает
3. Входная самооценка (Модуль 1) и выходная (Модуль 8) — сравнить
4. Собрать обратную связь, составить список исправлений
5. Внести исправления, зафиксировать финальную версию

## Критерии приёмки

- 2-3 добровольца прошли курс
- Реальное время ±30% от запланированного (8ч)
- Все блокеры (неработающие установки, ошибки в примерах) исправлены
- У каждого добровольца полный рабочий сетап после курса
