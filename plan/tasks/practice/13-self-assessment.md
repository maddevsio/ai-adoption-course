# Задача 13: Анкета самооценки уровня зрелости

## Цель

Создать анкету, которую ученик заполняет в начале (Модуль 1) и в конце (Модуль 8) курса для измерения прогресса.

## Что сделать

Анкета из 8-10 вопросов. Два применения:
- Модуль 1: входная диагностика (где я сейчас?)
- Модуль 8: выходная диагностика (что изменилось?)

### Вопросы

Адаптировать из `sources/Портрет ИИ зрелости.md`:
- Фастчек (раздел в конце документа)
- Вопросы для скрининга
- Матрица компетенций (упростить до самооценки)

Формат каждого вопроса: утверждение → выбор ответа (шкала или варианты).

Примеры направлений:
1. Какие AI-инструменты используете? (чат / копилот / агент / параллельные агенты)
2. Как часто? (эпизодически / ежедневно / постоянно)
3. Как формулируете запрос? (вопрос / задание с контекстом / спецификация)
4. Проверяете ли результат? (нет / иногда / всегда по чеклисту)
5. Знаете ли MCP? (нет / слышал / настроил / использую)
6. Работали с параллельными агентами? (нет / пробовал / регулярно)
7. Следите за трендами? (пассивно / читаю блоги / экспериментирую)
8. Стоимость запросов: знаете примерно? (нет / примерно / точно)

### Подсчёт

Простая шкала: каждый ответ = 1-5 баллов. Итоговый балл → уровень зрелости (1-5).

## Формат результата

MD-файл `course/module-1-why-ai/self-assessment.md` с анкетой и ключом подсчёта.

## Критерии приёмки

- 8-10 вопросов
- Простой подсчёт (не нужен калькулятор)
- Результат однозначно определяет уровень 1-5
- Можно заполнить за 5 минут
- Формулировки понятны человеку без опыта с ИИ
