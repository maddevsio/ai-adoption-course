# Задача 14: Инструкция по установке рабочего окружения

## Цель

Пошаговая инструкция по установке Claude Code CLI. Основной инструмент курса — терминальный агент.

## Что сделать

### Установка Claude Code CLI

1. Установить через npm или brew
2. Настроить API-ключ или подписку
3. Проверить: `claude --version`, тестовый запрос
4. Базовая настройка: permissions, default model

### Первый тест

- Открыть свой проект в терминале
- Запустить Claude Code
- Задача: "Напиши архитектурную схему этого проекта" — агент читает файлы, анализирует структуру, выдаёт схему
- Ожидаемый результат: ученик видит, что агент понимает проект

### Песочница для практики (если нет своего проекта)

Короткий рецепт (5 минут) для тех, у кого нет подходящего проекта:

```bash
mkdir task-manager && cd task-manager
git init
# Python вариант:
pip install fastapi uvicorn pytest httpx
# или TypeScript вариант:
# npx create-express-app task-manager
```

Результат: минимальный CRUD (GET/POST /tasks) на 3-5 файлов, работающие тесты. Намеренно без валидации, без PUT/DELETE, без AGENTS.md -- это "точки роста" для упражнений модулей 4-7.

Дать ученику промпт для агента: "Создай минимальный FastAPI-проект Task Manager с GET /tasks, POST /tasks, GET /tasks/{id}. SQLite. Pytest. Без валидации. Без PUT/DELETE."

### Дополнительно (опционально)

- Установка Cursor как альтернативного инструмента
- Настройка VS Code + Copilot как второго варианта

### Для всех вариантов

- Troubleshooting: 5 конкретных проблем:
  1. `npm ERR! EACCES` — проблема прав при глобальной установке (решение: nvm или --prefix)
  2. API-ключ не принимается / "Invalid API key" (решение: проверить формат, регион, план)
  3. `claude: command not found` после установки (решение: PATH, перезапуск терминала)
  4. Таймаут при первом запуске / "Connection refused" (решение: прокси, VPN, firewall)
  5. Агент не видит файлы проекта / "Permission denied" (решение: запускать из корня проекта, проверить permissions)
- Чеклист готовности (5 пунктов)

## Формат результата

MD-файл `course/module-2-tools/practice-setup.md`

## Критерии приёмки

- Фокус на Claude Code CLI как основном инструменте
- Работает на macOS и Linux
- Каждый шаг проверяем
- Первый тест: архитектурная схема проекта (не тривиальный "hello world")
- Разработчик пройдёт за 30-40 мин
- Конкретные версии из задачи 03
