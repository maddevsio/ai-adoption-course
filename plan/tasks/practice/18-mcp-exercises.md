# Задача 18: Упражнения по настройке MCP

## Цель

Практика для Модуля 6. Ученик устанавливает, настраивает и использует конкретные MCP-серверы.

## Что сделать

### Шаг 1: Установить обязательные MCP-серверы (20 мин)

Пошаговая инструкция для каждого:

1. **git MCP**: глубокая работа с git из агента
   - Установка, конфигурация
   - Проверка: агент анализирует git history проекта

2. **jira MCP**: ворклоги и задачи
   - Установка, конфигурация (API token)
   - Проверка: агент читает задачи из Jira
   - Альтернативы для тех, у кого нет Jira:
     - GitHub Issues MCP (https://github.com/modelcontextprotocol/servers/tree/main/src/github) — бесплатно, у всех есть GitHub
     - Linear MCP (если команда использует Linear)
     - Todoist/Notion MCP — как минимальная замена

3. **IDE MCP**: запуск дебаггера, настройка окружения
   - Установка для PyCharm / VS Code / другой IDE
   - Проверка: агент может взаимодействовать с IDE

4. **Figma MCP** (https://github.com/GLips/Figma-Context-MCP):
   - Установка, конфигурация (Figma token)
   - Проверка: агент читает дизайн из Figma
   - Примечание: если нет Figma — показать на демо-файле

### Шаг 2: Настроить конфигурацию (10 мин)

- Полный пример конфигурации с 3-4 серверами для Claude Code
- Полный пример для Cursor (если используется)
- Проверка: все серверы видны агенту

### Шаг 3: Подготовленная задача с MCP (15 мин)

Задание, демонстрирующее ценность MCP:
- Пример: "Посмотри в Jira задачу X, проанализируй git-историю связанных файлов, предложи план реализации"
- Или: "Возьми дизайн из Figma и сгенерируй компонент"
- Ученик видит, как агент использует несколько MCP-серверов для одной задачи

### Шаг 4: Ознакомление с расширенным каталогом (10 мин)

Ученик просматривает расширенный список и выбирает 1 сервер для своего проекта:
- mcp-mermaid — генерация диаграмм
- mcp-miro — работа с досками
- mcp-obsidian — заметки
- Puppeteer — web automation
- PostgreSQL — запросы к БД
- PowerPoint — презентации
- browserstack — тестирование

Устанавливает, настраивает, проверяет.

## Формат результата

MD-файл `course/module-6-mcp/practice.md`

## Критерии приёмки

- 4 обязательных MCP-сервера: git, jira, IDE, Figma
- Для каждого: установка, конфигурация, проверка
- Альтернативы для тех, у кого нет Jira/Figma
- Полные примеры конфигурации
- Расширенный каталог с описаниями
- Troubleshooting: 3-5 типичных проблем
- Суммарно 55 мин
